Preguntas frecuentes

¿Qué tecla es Command (⌘) en un teclado de Windows?

La tecla Windows (⊞). Al conectar un teclado de Windows a una Mac, la tecla Windows cumple la función de Command (⌘): copiar es Windows + C, pegar es Windows + V, guardar es Windows + S.

¿Qué tecla es Option (⌥) en un teclado de Windows?

La tecla Alt. En un teclado de Windows conectado a una Mac, Alt funciona como Option (⌥).

¿Command es lo mismo que Control en Mac?

No. En Mac, Command (⌘) hace lo que Ctrl hace en Windows (copiar, pegar, guardar). Control (⌃) en Mac es una tecla distinta que se usa para otros atajos, como cambiar el idioma del teclado.

¿Cómo hago una captura de pantalla en Mac?

Shift + Command + 3 captura toda la pantalla, Shift + Command + 4 captura una zona que elijas, y Shift + Command + 5 abre la herramienta con opción de grabar la pantalla.

¿Cómo cambio el idioma del teclado en Mac?

Con Control + Espacio (⌃ + Espacio) cambiás entre los idiomas de teclado que tengas configurados.

¿Puedo usar un teclado de Windows en una Mac?

Sí, funciona sin instalar nada. Solo cambian los nombres de las teclas modificadoras: la tecla Windows pasa a ser Command (⌘), Alt pasa a ser Option (⌥) y Ctrl sigue siendo Control (⌃).
