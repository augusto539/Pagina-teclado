Frequently asked questions

What key is Command (⌘) on a Windows keyboard?

The Windows key (⊞). When you connect a Windows keyboard to a Mac, the Windows key acts as Command (⌘): copy is Windows + C, paste is Windows + V, and save is Windows + S.

What key is Option (⌥) on a Windows keyboard?

The Alt key. On a Windows keyboard connected to a Mac, Alt works as Option (⌥).

Is Command the same as Control on a Mac?

No. On a Mac, Command (⌘) does what Ctrl does on Windows (copy, paste, save). Control (⌃) on a Mac is a separate key used for other shortcuts, such as switching the keyboard language.

How do I take a screenshot on a Mac?

Shift + Command + 3 captures the whole screen, Shift + Command + 4 captures a selected area, and Shift + Command + 5 opens the tool with an option to record the screen.

How do I switch the keyboard language on a Mac?

Press Control + Space (⌃ + Space) to switch between the keyboard languages you have configured.

Can I use a Windows keyboard on a Mac?

Yes, it works without installing anything. Only the modifier key names change: the Windows key becomes Command (⌘), Alt becomes Option (⌥), and Ctrl stays as Control (⌃).
